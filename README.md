MOVIE Explorer
A well designed movie explorer app that allows the user to explore the movies, be able to look at all the details by clicking on the movie and also be able to mark their favorites movies.

Technologies used:
-React
-javascript
-Tailwindcss.

Project structure:
/src
  /components
    Navbar.jsx
    MovieCard.jsx
    SearchBar.jsx
  /pages
    Home.jsx
    MovieDetails.jsx
    Favorites.jsx
  /hooks
    useFetchMovies.js
    useFavorites.js
  /utils
    api.js
  App.jsx
  main.jsx
  index.css (optional, for global styles)
